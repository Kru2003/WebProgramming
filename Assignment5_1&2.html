Background & Text Manipulation Example

This is a demonstration of how you can manipulate background and text properties using CSS.

In this example, we've set a light gray background for the body and added a white container with rounded corners and a subtle shadow effect to create a material-like card.

We've also styled the heading to be center-aligned, uppercase, and in a dark color. The paragraphs have a slightly lighter color and increased line height for better readability.

Finally, we've created a highlight class for emphasizing certain text by giving it a yellow background and dark text color.
